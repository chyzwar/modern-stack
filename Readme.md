# Modern Stack

Typescript Create React App + Fastify API in monorepo

## Getting started

Repo provide init script that will setup baseline env files
It will check if user have docker and docker-compose installed.

```sh
yarn init
```

Once project is initialized:

```sh
yarn start
```
